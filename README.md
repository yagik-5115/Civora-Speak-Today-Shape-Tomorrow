# 🚀 Civora

> Smart Civic Issue Reporting Platform for transparent and citizen-friendly governance.

---

# 📌 Overview

**Civora** is a modern civic-tech frontend platform designed to simplify public issue reporting and improve communication between citizens and local authorities.

Users can report issues like:

- Road damage
- Garbage overflow
- Water leakage
- Streetlight problems
- Public infrastructure complaints

The platform focuses on:

- Clean UI/UX
- Multilingual accessibility
- Voice-based reporting
- Real-time complaint tracking
- Reward-driven civic engagement

---

## 🌐 Multilingual Support
